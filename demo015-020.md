## 详细版本说明

### demo016 - 统一日志 + 请求响应日志（MDC + 拦截器）

- 引入 `LoggingFilter`（基于 `OncePerRequestFilter` + `ContentCachingRequestWrapper`）
- 实现**完整请求体 + 响应体**记录
- MDC 传递 traceId，全链路日志可串联
- 敏感字段自动脱敏（password、token）
- 日志格式专业化（logback-spring.xml）

**里程碑意义**：让系统具备了生产环境最基础的「可观测性」。

### demo017 - Redis 缓存集成（@Cacheable + 缓存穿透处理）

- 集成 `spring-boot-starter-data-redis`
- 自定义 `RedisTemplate`（Jackson 序列化）
- 使用 `@Cacheable` + `@CacheEvict`
- 手动实现**缓存穿透防护**：空值缓存 + 随机过期时间（防雪崩）
- 新增 `/students/protect/{id}` 专门用于演示防护效果

**里程碑意义**：掌握缓存这把「双刃剑」的正确用法。

### demo018 - JWT + Spring Security 基础认证

- 引入 Spring Security 6 + `java-jwt`
- 实现 `/auth/login` 签发 JWT
- `JwtAuthenticationFilter` 拦截所有请求并注入 SecurityContext
- 基于 `sys_user` 表做真实用户认证
- 角色支持（ADMIN / TEACHER / STUDENT）
- 自定义 `UserDetailsService` 消除默认密码警告

**里程碑意义**：从「谁都能调」进化到「必须带合法 Token」。

### demo019 - Docker + Docker Compose 一键部署

- 多阶段 `Dockerfile`（构建镜像极小 + 非 root 运行）
- 完整 `docker-compose.yml`（app + mysql + redis）
- 环境变量覆盖配置（Docker 内使用服务名）
- 健康检查 + readiness probe
- `.dockerignore` 优化构建

**里程碑意义**：项目具备了「开箱即用」的部署能力。

### demo020 - 完整项目重构（模块划分 + 代码规范）

- **DTO / VO 彻底分离**：Controller 只返回 VO，不再暴露 Entity
- 新增 `StudentVO`，Controller 增加转换逻辑（演示 MapStruct 雏形）
- 强化 `common/` 公共模块
- 为后续按 `module/` 领域拆分做好架构准备
- 整体目录结构和命名更加规范

**里程碑意义**：从「能跑」进化到「好维护、好扩展」。

## 下一步规划：真正的小型电商系统（Mini E-commerce）

经过 demo016-020 的基础设施锤炼，我们终于可以开始做**真正有业务价值的项目**了。

### 项目定位

**小型电商系统（Mini Mall）**，而非简单的 Todo / 学生系统。

核心模块（第一阶段 MVP）：
- 用户体系（注册、登录、JWT、地址管理）
- 商品体系（分类、SPU、SKU、库存）
- 购物车
- 订单系统（下单事务、库存扣减、订单状态流转）
- 简单支付模拟

### 技术栈（最终版）

**后端**（基于 demo020 架构继续演进）：
- Spring Boot 3 + MyBatis-Plus
- Redis（缓存 + 分布式锁）
- JWT + Spring Security
- Docker + Docker Compose
- 后续可加入：RabbitMQ / RocketMQ、Seata、Sentinel 等

**前端**（全新引入）：
- Vite + Vue 3 + TypeScript
- Pinia + Vue Router
- Element Plus（或 Ant Design Vue）
- Axios + 统一拦截器
- 后期考虑：后管系统 + C 端 H5

### 推荐演进路线（demo021 之后）

| 阶段       | 主题                        | 核心交付                                      |
|------------|-----------------------------|-----------------------------------------------|
| v1         | 基础骨架 + 用户 + 商品      | 登录注册、商品列表/详情、SKU 展示             |
| v2         | 购物车 + 下单               | 购物车 CRUD、下单事务（扣库存）               |
| v3         | 订单 + 支付模拟             | 订单列表、状态流转、支付回调模拟              |
| v4         | 后台管理系统                | 商品维护、订单管理、用户管理（Vue3 后管）     |
| v5         | 性能与稳定性                | Redis 缓存策略、分布式锁、限流、监控          |
| v6         | 部署与工程化                | Docker Compose 生产版、CI/CD、Nginx 反向代理  |
